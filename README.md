# Iob Bank - Front End

Este é um projeto React + NextJS, responsável por consumir

## Requisitos

- NodeJS
- Yarn

## Como rodar

1. Dentro da pasta do projeto, execute: `yarn`
2. Após instalado as dependências, execute: `yarn dev`
3. Abra o navegador e acesse a url `http://localhost:3000`
